C 社团招新

Solution

贪心，先将所有人放到最想去的社团。若某个社团人数大于 n2\frac{n}{2}2n​（设为 kkk 号社团），则继续处理。

设编号为 iii 的人对社团的满意度从小到大分别是 ai1a_i^1ai1​、ai2a_i^2ai2​、ai3a_i^3ai3​。按照 ai3−ai2a_i^3-a_i^2ai3​−ai2​ 的大小，对所有人进行排序。

从小到大枚举人，若 ai3a^3_iai3​ 代表的是第 iii 个人对 kkk 号社团的满意度，则答案减去 ai3a_i^3ai3​ 加上 ai2a_i^2ai2​。直到社团 kkk 人数小于 n2\frac{n}{2}2n​ 为止。

时间复杂度 O(nlog⁡n)O(n\log n)O(nlogn)。

E 成绩统计

Solution

注意到检查的同学越多，越可能满足条件。故我们对检查的同学数量 xxx 进行二分。

因为数字互相越接近，方差越小，所以对 vvv 的区间 [1,x][1,x][1,x] 进行排序，并枚举连续 kkk 个数，判断是否满足条件。

题目条件为

∑i=1k(vi−vˉ)2k<T\frac{\sum_{i=1}^k (v_i - \bar{v})^2}{k} < T k∑i=1k​(vi​−vˉ)2​<T

即

∑i=1k(vi−vˉ)2<kT\sum_{i=1}^k (v_i - \bar{v})^2 < k T i=1∑k​(vi​−vˉ)2<kT

∑i=1kvi−kvˉ2<kT\sum_{i=1}^k v_i - k\bar{v}^2 < k T i=1∑k​vi​−kvˉ2<kT

将 vˉ=∑i=1kvik\bar{v} = \frac{\sum_{i=1}^k v_i}{k}vˉ=k∑i=1k​vi​​ 代入得

F 网络流量监控

Solution

模拟题。

每条路径都按照 / 分割为字符串数组。

对每次询问，枚举恶意路径一个个匹配。注意到通配符只可能出现一次，对恶意路径分类讨论：

  1. 不含通配符 *，**：直接判断两条路径是否相同。
  2. 含通配符 *：先判断长度是否一致，然后逐位检查字符串是否相同。
  3. 含通配符 **：** 将恶意路径分割成前缀和后缀。分别判断恶意路径的前缀与后缀和询问路径的前缀与后缀是否相同。

G 上升序列构造

Solution

贪心 + 构造。要在保证后一个数大于前一个数的同时，后一个数尽可能小。

数字的位数可能会到 10500010^{5000}105000 级别，考虑转化成字符串处理。

设前一个字符串（已经加 0）为 aaa，长度 lal_ala​。当前字符串为 bbb，长度 lbl_blb​。分类讨论：

  1. 若 la<lbl_a<l_bla​<lb​：不用加 0。
  2. 若 la=lbl_a=l_bla​=lb​ 且 a<ba<ba<b ：不用加 0。
  3. 否则，枚举 iii 计算要使得 aaa 长为 iii 的前缀与 bbb 的前缀相同，需要加多少个 0 改造 bbb 的前缀，且改造后 bi+1>ai+1b_{i+1}>a_{i+1}bi+1​>ai+1​ 成立。若存在 iii 使得不等式成立，则在改造后的 bi+1b_{i+1}bi+1​ 后插入剩余的 0 使得 la=lbl_a=l_bla​=lb​。否则在 bbb 的第二位开始插入若干个 0，使得 lb=la+1l_b=l_a+1lb​=la​+1。

时间复杂度 O(n2)O(n^2)O(n2)。
