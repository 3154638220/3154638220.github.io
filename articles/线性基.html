线性基

2024-09-14 约 2 分钟

Link

【模板】线性基 - 洛谷

Idea

易知给出的正整数和它们任意异或后的所有结果和 000 组成的集合 GGG 关于异或运算封闭，且为有限集合，满足结合律，有单位元和逆元。所以 GGG 构成一个群。我们求的线性基是这个群的最小线性基，即 GGG 中的所有元素均可被基中的元素线性表出。

构造线性基的方法详见 insert 函数。

要找到 GGG 中最大的异或和，就在线性基中由高位到低位尝试异或出最大的结果，贪心可得。

查询某数是否可被 GGG 中元素异或得到，也是在线性基中尝试从高位到低位构建。

Code

#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

const ll N = 1e5 + 10;

ll p[N], n, a[N], ans;

void insert(ll x) 
{
    for (int i = 63; ~i; i--) {
        if (!(x >> i)) 
            continue;
        if (!p[i]) {
            p[i] = x;
            return;
        } else 
            x ^= p[i];
    }
}

int main()
{
    scanf("%lld", &n);
    for (int i = 1; i <= n; i++) {
        scanf("%lld", &a[i]);
        insert(a[i]);
    }
    for (int i = 63; ~i; i--) 
        ans = max(ans, ans ^ p[i]);
    printf("%lld", ans);
    return 0;
}
