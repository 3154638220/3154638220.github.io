  1. 当 sl=srs_l=s_rsl​=sr​ 时，执行 l+1l+1l+1，r−1r-1r−1，意味着两个字符已经匹配。

Solution

用栈模拟括号匹配。遍历括号串，遇到 ( 压入栈中，遇到 ) 弹出栈顶。

显然，叶节点的特征是它的左右两半括号在括号序列中相邻，且它与根的距离等于它匹配成功并弹出后，栈的剩余元素个数。

D

Solution

  • 一个子串是 444 的倍数，当且仅当最后两位组成的十位数是 444 的倍数。

  • 一个子串是 555 的倍数，当且仅当末位是 000 或 555。

因此子串是否合法与前缀无关。枚举子串的末位位置 iii ，只要符合上述条件，对答案的贡献就至少是前缀的数量 i−1i-1i−1。

最后特判长度为 111 的子串情况即可。

E

Solution

形式化题意：对给定的正整数 nnn，找最小的 kkk，使得 n=2k−1+pn=2^k-1+pn=2k−1+p。其中 ppp 为素数或 000。

移项得 p=n−2k+1p=n-2^k+1p=n−2k+1。注意到 n≤105n\le 10^5n≤105，kkk 不会很大。枚举 kkk 并判断 n−2k+1n-2^k+1n−2k+1 是否为素数即可。

Solution

数位 dp。设 pip_ipi​ 表示按题意去掉一种数字后，位数小于等于 iii 的数有多少个。由乘法原理，每一位都有 999 种选择，pi=9ip_i=9^{i}pi​=9i。

设 nnn 第 iii 位上的数为 aia_iai​。从高位到低位依次统计答案。对于第 iii 位，我们需要计算比 aia_iai​ 小的合法数字有多少个，这些数字可以作为当前位，且其后的 i−1i-1i−1 位可以任意填合法数字。

由于数字 xxx 被去掉了：

  • 若 ai<xa_i < xai​<x，在 0 到 ai−1a_i-1ai​−1 中没有数字被去掉，共有 aia_iai​ 种选择，对答案的贡献为 pi−1×aip_{i-1} \times a_ipi−1​×ai​。
  • 若 ai>xa_i > xai​>x，在 0 到 ai−1a_i-1ai​−1 中数字 xxx 被去掉了，共有 ai−1a_i - 1ai​−1 种选择，对答案的贡献为 pi−1×(ai−1)p_{i-1} \times (a_i - 1)pi−1​×(ai​−1)。

将每一位的贡献累加即可得出 nnn 前面有多少个数。由于题目要求的排名从 1 开始（即自然数 0 排在第 1 位），最终的累加结果需要加 1。
